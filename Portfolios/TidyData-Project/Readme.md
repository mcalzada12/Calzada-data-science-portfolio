# 🥇 Olympic 2008 Medalists Data Set 

## 📊 About the Project 

This project focuses on transforming a raw dataset of 2008 Olympic medalists into a tidy format and performing exploratory data analysis (EDA). The main objective is to apply tidy data principles to restructure the dataset and generate meaningful insights through visualization and aggregation.
---
But what exactly is data cleaning? 
- Data Cleaning is the process used for specific sets of inofrmation in order for them to be easier to work with, visualize and present.
